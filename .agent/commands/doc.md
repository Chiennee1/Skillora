# Command: /doc

> Generate documentation for Skillora.

## Syntax
```
/doc [--api] [--readme] [--setup]
```

## Execution

1. Scan project structure
2. Generate/update requested documentation
3. For `--api`: Generate OpenAPI docs or update `context/api-catalog.md`
4. For `--readme`: Generate README.md
5. For `--setup`: Generate SETUP.md with local environment instructions
